# QOG Front End

## How to run the code

```npm install``` (Run only once)

```npm start```

It will serve on local: http://localhost:3000/

## How is the page served

Via [Github pages](https://pages.github.com)

We've already configured. Just deploy code to `gh-pages` branch, and Github will catch up the changes shortly

## How to deploy to Github pages

```npm run deploy```

It will automatically build and commit to `gh-pages` branch
